# ⚡ 市長請決定：全城大停電！

> **「當資源有限時，你該如何選擇？」** > 這是一個專為國小社會科設計的互動決策遊戲，模擬城市在下午 14:00 遭受雷雨襲擊後的斷電危機。

## 📖 遊戲背景
下午 14:00，一場突如其來的強大雷雨擊中了都市電網，全城陷入大停電！身為市長，你手中僅剩下 **100%** 的緊急備用電力。

交通號誌、醫療設施、校園安全、民生物資... 每一項都需要電力。你必須在有限的電力耗盡前，從 20 個隨機發生的緊急情境中做出 3 個關鍵抉擇。

## 🌟 核心教學價值
本遊戲旨在引導學生思考以下社會科核心概念：
* **資源分配**：理解電力（資源）是有限的，無法滿足所有需求。
* **機會成本**：學會分析每一個選擇背後所「犧牲」掉的事物（例如：選擇救疫苗，可能就得延後解救受困電梯的人）。
* **決策責任**：強調身為公民或領導者，做出選擇時必須極其「謹慎」，並對結果負責。

## 🛠️ 技術規格
* **語言**：HTML5, CSS3, JavaScript (純原生代碼，無須安裝依賴)。
* **設計**：響應式網頁設計 (RWD)，適配手機、平板與電腦。
* **核心功能**：
    * 20 題隨機題庫系統。
    * 電力點數透明化標示。
    * 即時後果回饋分析報告。
